## Start in under 2 minutes

```bash
  npm install -g @bvbmz/local-proxy
```

Or run instantly with `npx`:

```bash
  npx @bvbmz/local-proxy --target https://api.example.com
```

```bash
  # 1) scaffold config
  local-proxy --init
  # 2) run proxy
  local-proxy --target https://api.example.com
```

## Default runtime

- Port: `5050`
- API prefix: `/api`
- Scenarios file: `./scenarios.json`

Point your frontend to:

```bash
  http://localhost:5050/api
```

## Step-by-step flow

<ol>
  <li>Install `@bvbmz/local-proxy`.</li>
  <li>Run `local-proxy --init` to generate `scenarios.json`.</li>
  <li>Start with `local-proxy --target &lt;your-api-url&gt;`.</li>
  <li>Switch `active_scenario` values in `scenarios.json` while developing.</li>
</ol>

## Environment variables

You can configure runtime with env vars:

- `TARGET`
- `PORT`
- `API_PREFIX`

CLI flags take precedence over env vars.
